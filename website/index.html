Setup

Two minutes to a living graph

Install, wire into Claude Code, forget about it. Your graph builds itself from there.

01

Install the CLI

One command. Works on macOS, Linux, and Windows. Requires Python 3.12+ and uv.

$ uv tool install "git+https://github.com/akashub/nexus.git#subdirectory=backend"
02

Wire into Claude Code

Installs MCP server, skill file, and session hooks. Claude reads and writes your graph automatically.

$ nexus mcp install
  MCP server registered
  Skill installed · Hooks registered
03

Seed your graph

Scan a project to populate from package files. Or just start working — hooks capture everything.

$ nexus scan . --enrich
  Found 47 dependencies in package.json, pyproject.toml
  Enriching React... Enriching FastAPI... done
04

Explore

Desktop app or CLI. Search, visualize, ask questions, detect gaps, track your journey.

$ nexus journey

Week of Apr 28
  ├── React [framework]
  ├── Tailwind [framework]
  └── Playwright [devtool]

Ecosystem

Works with your tools

Plugs into the AI coding workflow you already have.

Claude Code

MCP server for real-time graph access. Hooks capture installs. Skill teaches the ledger format.

Codex & MCP Clients

Any MCP-compatible tool works. Register nexus mcp serve and go.

Ollama

Gemma 4B for generation, nomic-embed-text for embeddings. Local, no API keys.

Cloud AI

Anthropic or OpenAI for higher-quality enrichment. One env var, add --model cloud.

Desktop App

Tauri-based native app. Interactive graph, search, side panels, project switching.

Context7

Real library docs for enrichment. Falls back to PyPI, npm, GitHub, libraries.io.

How It Works

Two minutes to a living knowledge graph

Install, wire into Claude Code, and forget about it. Your graph builds itself.

01

Install

One command installs the CLI globally. Works on macOS, Linux, and Windows. Requires Python 3.12+ and uv.

$ uv tool install "git+https://github.com/akashub/nexus.git#subdirectory=backend"
02

Wire into Claude Code

Installs the MCP server, skill file, and session hooks. Claude Code now reads and writes your graph.

$ nexus mcp install
  MCP server registered
  Skill installed
  Hooks registered
03

Seed your graph

Scan an existing project to populate your graph from package files. Or just start working — hooks capture everything.

$ nexus scan . --enrich
  Found 47 dependencies in package.json, pyproject.toml
  Enriching React... Enriching FastAPI... done
04

Explore

Open the desktop app or use the CLI. Search, visualize, ask questions, detect gaps, see your learning journey.

$ nexus journey

Week of Apr 28

  ├── Tailwind [framework] — Utility-first CSS framework
  └── Playwright [devtool] — End-to-end testing

Integrations

Works with your tools

Nexus plugs into the AI coding workflow you already have.

Claude Code

MCP server gives Claude real-time graph access. Hooks capture installs. Skill teaches the ledger format.

Codex & MCP Clients

Any tool that supports MCP servers works. Register nexus mcp serve and go.

Ollama (Local AI)

Gemma 4B for generation, nomic-embed-text for embeddings. Runs on your machine, no API keys needed.

Cloud AI (Optional)

Anthropic or OpenAI for higher-quality enrichment. Set one env var and add --model cloud.

Desktop App

Tauri-based native app with interactive graph, search (Cmd+K), side panels, and project switching.

Context7

Fetches real library docs for enrichment. Falls back to PyPI, npm, GitHub, and libraries.io.

Start mapping your knowledge

Quick Start
$ uv tool install "git+https://github.com/akashub/nexus.git#subdirectory=backend"
$ nexus mcp install
$ nexus db init

# Optional: local AI models
$ ollama pull gemma4
$ ollama pull nomic-embed-text

# Seed from your project
$ nexus scan . --enrich
